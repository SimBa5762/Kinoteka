
# Kinoteka

Щоб запустити проєкт, необхідно завантажити його на свій пристрій. Для цього виконайте наступні кроки:

1. Натисніть на кнопку **Code** на сторінці репозиторія.
2. Скопіюйте посилання.
3. Відкрийте термінал і перейдіть у папку, в яку хочете клонувати проєкт, командою:  
   `cd шлях/до/папки`
4. Виконайте в терміналі команду для клонування:  
   `git clone скопійоване_посилання`
5. Відкрийте у редакторі коду папку з проєктом.
6. Встановіть необхідні залежності (модулі) командою:  
   `npm install`
7. Запустіть проєкт командою у терміналі:  
   `node server`

У проєкті виконано усі поставлені рівні, а саме від 1. "Каталог" до 4. "Повний CRUD".

---

## Опис структури БД

В базі даних реалізовані наступні таблиці: жанри, користувачі, фільми, відгуки. Кожна таблиця має унікальний ідентифікатор `id` для забезпечення зв'язків між сутностями.

* **Жанри (`genres`)** — зберігаються назва жанру і ідентифікатор для поєднання з іншими таблицями.
* **Фільми (`movies`)** — містить назву, посилання на фотографію, рік, ідентифікатор жанру (зовнішній ключ), опис і оцінку.
* **Користувачі (`users`)** — містить ім'я користувача, його пошту, хэш паролю та роль для розподілення funktionальних можливостей за рівнем доступу.
* **Відгуки (`reviews`)** — містить ідентифікатори користувача (що його залишив) і фільму (до якого він написаний), текстовий огляд, оцінку і дату створення.
