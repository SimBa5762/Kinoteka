# Kinoteka

Щоб запустити проект необхідно завантажити його на свій пристрій, 
для цього виконати наступні кроки:
натиснути на кнопку code на сторінці репозиторія
скопіювати посилання
відкрити термінал і перейти у папку в яку хочете клонувати проект командою: cd шлях/до/папки
виконати в своєму терміналі команду: git clone скопійоване опсилання
відкрити у редакторі коду папку з проектом
запустити проект командоюу терміналі: node server 

у проекті виконано усі поставлені рівні, а саме від 1. "Каталог" до 4. "Повник CRUD"

Опис структури бд:

в базі даних реалізовані наступні таблиці: жанри, користувачі, фільми, відгуки
в таблиці жанри зберігаються назва жанру і ідентифікатор для поєднання з іншими таблицями, він наявний у кожній таблиці
таблиця фільми містить назву, посилання на фотографію, рік ідентифікатор жанру, опис і оцінку
таблиця користувачі ім'я користувача, його пошта, пароль і роль для розподілення функціональних можливостей за рівнем доступу.
таблиця огляду містить ідентифікатори користувача, що його залишив, і фільму до якого він написаний, оцінку і дату створення

схему таблиці і типи зв'язків між ними зображено на рис. 1.
